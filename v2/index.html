    音游功能

    舞萌DX

    参数组说明：
    ver 版本，可用dx或标准
    diff 难度，可用绿黄红紫白
    rating 谱面定数
    level 谱面等级
    title 曲目曲名
    id 谱面ID
    alias 别名
    label 投票标签
    bpm 节拍
    min 最小值
    max 最大值
    name 名字
    score 分数
    @ 群员或玩家名称
    player 玩家名称
    page 页数
    xxx 随意文字
    plate 牌子或将牌
    grade 评价等级或fc，可用fc、ap、SSS、aaa等
    plan 进度，可用已完成、未完成、未游玩

    • 今日舞萌 今日舞萌运势
    • 随个 随机发送一个谱面
      参数：<ver> <diff> <rating>
    • 查歌 曲名含该文字的曲目
      参数：[title]
    • 是什么歌 查询该别名的曲目
      参数：[alias]
    • 有什么别名 查询该曲目的所有别名
      参数：[id / alias / title]
    • 定数查歌 使用定数查歌或定数的上限和下限查询
      参数：[rating] {page} / [min] [max]
    • bpm查歌 使用BPM查歌或BPM的上限和下限查询
      参数：[bpm] {page} / [min] [max]
    • 曲师查歌 查询指定曲师的所有曲目
      参数：[name]
    • 谱师查歌 查询指定谱师的所有曲目
      参数：[name]
    • id 查询乐曲信息或谱面信息
      参数：[id]
    • 分数线 查询分数线，详细说明请发送[分数线 帮助]
      参数：[diff]+[id] [score]
    • 猜歌/猜曲绘 猜歌游戏
    • mai猜歌 开启或关闭猜歌功能
      参数：[on / off]
    • 重置猜歌 重置当前猜歌进程
    • minfo 查询指定曲目成绩
      参数：<@> [id/alias/title]
    • ginfo 查看指定曲目游玩总览，默认为紫谱
      参数：<diff> [id/alias/title]
    • b50 查询b50
      参数：<@>
    • 定数表 查看指定等级定数表
      参数：[level]
    • 完成表 查看指定等级游玩完成表
      参数：[level] <grade>
    • 查看排行 使用页数或玩家名查看ra排行
      参数：[page/@]
    • <xxx>mai<xxxx>什么 随机一首歌，可带上分字样
      示例：今天mai出勤打什么分
    • 我要在[diff]上[score]分<@> 推荐的上分乐曲
      示例：我要在13+上1分
    • [plate]进度<@><page> 牌子完成进度
      示例：霸者进度 @柚子
    • [rating][grade]<plan>进度<@><page> 等级评价完成进度
      示例：13+SSS进度 | 13+S已完成进度
    • [rating]分数列表<@> 等级分数列表
      示例：13+分数列表

    娱乐功能

    • 猜单词/wordle 进行猜单词游戏
    游戏介绍和功能介绍：
绿色块代表此单词中有此字母且位置正确；
黄色块代表此单词中有此字母，但该字母所处位置不对；
黄色块的个数代表 此单词中有这个字母的数量 个数；
灰色块代表此单词中没有此字母；
猜出单词或用光次数则游戏结束。
可发送“结束”结束游戏；可发送“提示”查看提示。
可使用 -l / --length 指定单词长度，默认为 5
可使用 -d / --dictionary 指定词典，默认为 CET4
支持的词典：GRE、考研、GMAT、专四、TOEFL、SAT、专八、IELTS、CET4、CET6
    • 猜成语/handle
    子功能：
结束/结束猜成语/结束游戏
规则:
你有十次的机会猜一个四字词语；
每次猜测后，汉字与拼音的颜色将会标识其与正确答案的区别；
汉字的右上角标的数字1、2、3、4分别对应一、二、三、四声；
青色 表示其出现在答案中且在正确的位置；
橙色 表示其出现在答案中但不在正确的位置，至多着色 **此单词中有这个字母的数量** 次；
每个格子的 汉字、声母、韵母、声调 都会独立进行颜色的指示。
当四个格子都为青色时，你便赢得了游戏！
可发送“结束”结束游戏；可发送“提示”查看提示。
